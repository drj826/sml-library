>>>configuration_item.ci-000006

title:: SML Semantics

type:: configuration file

is_part_of:: ci-000001

description:: [a:SML] semantics specify the terms that represent the
entities defined for an SML library, the entity properties, the
relationships that hold among entities, and the formal axioms that
constrain the interpretation and well-formed use of these terms.  The
semantics that hold for an SML library are expressed as an ontology in
one or more ontology configuration files.
