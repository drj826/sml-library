>>>requirement.rq-000381

title:: Define the Syntax for an Outcome Element

type:: data format

is_part_of:: rq-000372

description:: The data format shall define the syntax for an outcome
element.  The purpose of an outcome element is to declare the status
value of an entity as determined by a test or audit.  The outcome
information shall include: entity tested (or audited), date of test
(or audit), status determined by test (or audit), and a description
(not to exceed one paragraph) of the outcome.
