>>>requirement.rq-000132

title:: Resolve Templates

is_part_of:: rq-000105

priority:: routine

type:: technical

description:: The software shall resolve template statements.
Templates are re-usable SML files containing variable tags.  The
template statement passes values for those variables.  The software
must replace the variable tags with the values and replace the
template statement with the template containing the replaced
variables.

index:: [id:rq-000132]; resolve templates
