>>>requirement.rq-000018

title:: Represent Pre-Formatted Text

priority:: routine

is_part_of:: rq-000326

type:: data format

description:: The data format shall have the capability to represent
pre-formatted text.  Pre-formatted text is verbatim text inserted into
a document while preserving all white spaces and line breaks.  This
requirement serves as a container for data format requirements related
to representing pre-formatted text.

index:: [id:rq-000018]; pre-formatted text
