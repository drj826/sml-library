>>>requirement.rq-000018

title:: Define the Syntax for a Division of Pre-Formatted Text

priority:: routine

is_part_of:: rq-000326

type:: data format

description:: The data format shall define the syntax for a division
of pre-formatted text.  Pre-formatted text is verbatim text inserted
into a document while preserving all white spaces and line breaks.

index:: rq-000018; pre-formatted text syntax; syntax!pre-formatted
text

No [g:SML:element]s should be allowed within a pre-formatted text
division because all text should be interpretted as pre-formatted.  If
you need elements like title and description consider using a
[g:SML:LISTING] instead.

outcome::2016-03-02:rq-000018:green: This requirement is
complete. (Don Johnson)
