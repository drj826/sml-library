>>>requirement.rq-000417

title:: Define the Syntax for a Status Reference String

is_part_of:: rq-000407

type:: data format

description:: The data format shall have the capability to represent
an inline status reference.  An entity may have a status property that
represents the degree to which the entity is complete.  The status
reference will enable authors to refer to the status of the entity
from anywhere in the library.
