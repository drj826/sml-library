title:: Validate Test Documents

is_part_of:: rq-000200

priority:: routine

type:: functional

description:: The software shall enable a user to validate test
documents.  Validation means checking the document to ensure it
conforms to a specified set of rules.  Test validation rules may
specify that every test case is traceable to requirements or
configuration items, or has specified required attributes.
