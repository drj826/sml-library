>>>requirement.rq-000338

title:: Gather Every Division of Every Parsed File to a Containing
Division or to the Document Itself

is_part_of:: rq-000128

priority:: routine

type:: technical

attr:: validation method = inspection

description:: The software shall gather every division of every parsed
file to a containing division or to the document itself.  The purpose
of this requirement is to ensure the data gathered into memory from
parsed files can be used to re-write those files back out to permanent
storage.
