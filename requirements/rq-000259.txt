>>>requirement.rq-000259

title:: Notify Users of First Order Semantics Errors

revision:: $Revision: 11687 $

date:: $Date: 2012-12-06 13:59:01 -0700 (Thu, 06 Dec 2012) $

author:: $Author: don.johnson $

is_part_of:: rq-000106

type:: technical

description:: The software shall notify users of errors in document
first order semantics.  The purpose of this requirement is to make it
easy for users to find and fix semantics errors in SML document files.
Semantics error notifications should specify the file and line number
of the error so the user knows exactly where to look for the problem.
Error notifications should also provide an informative message that
describes the error.

index:: [id:rq-000259]; notification!semantics error

Examples of possible first order semantics errors include:

+ Explicit declaration of infer-only property

+ Missing required property

+ Unknown division property

+ Invalid property cardinality

+ Invalid entity relationship
