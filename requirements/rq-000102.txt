title:: Format Output Text From Document Structure Objects

is_part_of:: rq-000199

type:: functional

description:: The software shall format output text from document
[g:SML2:structure] objects. Document structure objects are memory
resident objects created by the parser that represent document
structures like paragraphs, lists, sections, tables, figures, and
listings.  This requirement serves as a container for requirements
calling for formatting specific SML document structure features.
