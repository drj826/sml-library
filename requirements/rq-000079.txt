>>>requirement.rq-000079

title:: Represent a Glossary Reference

priority:: routine

is_part_of:: rq-000120

type:: data format

description:: The data format shall have the capability to represent a
glossary reference.  In other words, the data format must have the
capability to indicate that a word or phrase is in the glossary.  The
data format must allow glossary references to span newline characters.

index:: rq-000079; glossary references
