>>>requirement.rq-000010

title:: Define the Syntax for a Paragraph

priority:: routine

is_part_of:: rq-000327

type:: data format

description:: The data format shall define the syntax for a paragraph.
A paragraph is an element of document [g:SML1:structure], a
[g:SML:block] of [g:SML:text] that communicates an idea. The data
format representation of a paragraph must allow but not mandate
[g:SML:newline] characters within a paragraph.

index:: rq-000010; paragraph

<<<requirement
