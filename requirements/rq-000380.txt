>>>requirement.rq-000380

title:: Define the Syntax for a Script Element

type:: data format

is_part_of:: rq-000372

description:: The data format shall define the syntax for a script
element.  A script element instructs the application to run a script
and include the results in the document. The syntax shall have the
ability to indicate that the script results should be included as a
section as a certain depth.  The syntax shall capture the script
command.

index:: rq-000380; script element; element!script

outcome::2016-03-05:rq-000380:green: This requirement is
complete. (Don Johnson)
