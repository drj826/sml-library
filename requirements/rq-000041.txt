>>>requirement.rq-000041

title:: Represent Document Font Size

priority:: routine

is_part_of:: rq-000122

type:: data format

description:: The data format shall have the capability to represent
the document font size.  The document font size is the size of normal
narrative text.  The font size used for footnotes and other structures
may be relative to the document font size.

index:: rq-000041; document font size; font size!document
