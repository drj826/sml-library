>>>requirement.rq-000378

title:: Define the Syntax for a Plugin Element

type:: data format

is_part_of:: rq-000372

description:: The data format shall define the syntax for a plugin
element.  A plugin element instructs the application to run a plugin
and to put the results in the document. The syntax shall have the
ability to indicate that the plugin results should be included as a
section at a certain depth.  The syntax shall capture the plugin name
and any arguments to be passed to the plugin.

index:: rq-000378; plugin element; element!plugin

outcome::2016-03-05:rq-000378:green: This requirement is
complete. (Don Johnson)
