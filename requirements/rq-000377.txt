>>>requirement.rq-000377

title:: Define the Syntax for an Include Element

type:: data format

is_part_of:: rq-000372

description:: The data format shall define the syntax for an include
element.  The purpose of an include element is to instruct the
application to include a division from another part of the library.
This enables users to re-use content. The syntax shall have the
capability to indicate that the included text should be included as a
section at a certain depth.  The syntax shall capture the ID of the
division to be included.

index:: rq-000377; include element; element!include

outcome::2016-03-05:rq-000377:green: This requirement is
complete. (Don Johnson)
