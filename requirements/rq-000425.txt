>>>requirement.rq-000425

title:: Define the Syntax for a ``The Revision'' Reference String

type:: data format

is_part_of:: rq-000419

associated:: rq-000422

description:: The data format shall define the syntax for a ``the
revision'' reference string.  The purpose of a ``the revision''
reference string is to display the current revision number of the
library.  The revision number is a number automatically generated by a
revision control system, not a version number assigned by an author.
This is often used in a header or footer.
