title:: Represent an Inline Acronym Reference

priority:: routine

is_part_of:: rq-000334

type:: data format

description:: The data format shall have the capability to represent
an inline acronym reference.  In other words, the data format must
have the capability to indicate an acronym is in the official list of
acronyms.  The data format must allow acronym references to span
newline characters.
