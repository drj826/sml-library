>>>requirement.rq-000183

title:: Permit Library Ontology Configuration

revision:: $Revision: 11687 $

date:: $Date: 2012-12-06 13:59:01 -0700 (Thu, 06 Dec 2012) $

author:: $Author: don.johnson $

is_part_of:: rq-000182

priority:: routine

type:: non-functional

description:: The [g:SML:library] [g:SML:ontology] shall be
configurable in specified ways.  The [g:SML:ontology] applies to all
[g:SML:document]s in a [g:SML:library].

index:: [id:rq-000183]; ontology!configurability

Users need the ability to configure the ontology in the following
ways:

+ Declare entities

+ Declare entity properties by name

+ Declare whether or not an entity property is required

+ Declare the data type of an entity property

+ Declare inverse entity property implication rules

+ Declare the cardinality of an entity property

+ Declare whether an entity property is an 'imply only' property

+ Declare the order in which the property should appear in
  traceability matrices
