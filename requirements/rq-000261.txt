>>>requirement.rq-000261

title:: Publish a Static HTML Document Title Page

is_part_of:: rq-000001          # Publish an SML Document to Static HTML

type:: functional

description:: The software shall publish a static HTML document title
page.  The title page shall present the: (1) document title, (2)
subtitle, (3) name of author(s), (4) name of editor(s), (5) name of
translator(s), (6) name and location of the publisher, and (7) number
of the edition.

index:: rq-000261; title page; document title page

outcome::2016-03-20:rq-000261:green: This requirement is
complete. (Don Johnson)
