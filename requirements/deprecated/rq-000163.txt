>>>requirement.rq-000163

title:: Represent an Option to Persist Generated Content In A File

priority:: routine

is_part_of:: rq-000093

type:: data format

description:: The data format shall have the capability to represent
the author's choice that generated content should persist in a
specified file. The default behavior is to insert the generated
content into the output document but ~~not~~ persist it anywhere.

index:: [id:rq-000163]; generated content!persistent
