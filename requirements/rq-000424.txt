>>>requirement.rq-000424

title:: Define the Syntax for a ``The Version'' Reference String

type:: data format

is_part_of:: rq-000419

description:: The data format shall define the syntax for a ``the
version'' reference string.  The purpose of a ``the version''
reference string is to display the current version identifier of the
library.  The version identifier is assigned by the [title:ro-000004]
and not automatically generated by any revision control system.  This
is often used in a header or footer.

index:: rq-000424; the version reference; string!the version
reference; version identifier

outcome::2016-03-08:rq-000424:green: This requirement is
complete. (Don Johnson)
