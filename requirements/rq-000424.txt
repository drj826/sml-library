>>>requirement.rq-000424

title:: Define the Syntax for a ``The Version'' Reference String

type:: data format

is_part_of:: rq-000419

associated:: rq-000422

description:: The data format shall define the syntax for a ``the
version'' reference string.  The purpose of a ``the version''
reference string is to display the current version number of the
library.  The version number is a number assigned by an author and not
automatically generated by a revision control system.  This is often
used in a header or footer.
