>>>requirement.rq-000347

title:: Detect Circular Resource Use

is_part_of:: rq-000105

priority:: routine

type:: technical

associated:: rq-000181

description:: The software shall detect circular resource use.
Circular resource use occurs when a resource uses itself or any
ancestor in a chain of used resources.  This condition must be
detected to prevent infinite processing loops.

index:: [id:rq-000347]; circular resource use
