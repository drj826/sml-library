title:: Represent an Inline External Hyperlink

revision:: $Revision: 10973 $

date:: $Date: 2012-10-27 14:46:35 -0600 (Sat, 27 Oct 2012) $

author:: $Author: don.johnson $

priority:: routine

is_part_of:: rq-000334

type:: data format

description:: The data format shall have the capability to represent
an inline external hyperlink.  An inline external hyperlink enables
authors to provide a link to a specified type of resource external to
the document.

index:: [id:rq-000021]; external hyperlinks; hyperlink!external

The data format must enable authors to establish external hyperlinks
to the following types of resources:

+ web sites

+ E-mail addresses
