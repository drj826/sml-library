>>>requirement.rq-000382

title:: Define the Syntax for an Review Element

type:: data format

is_part_of:: rq-000372

description:: The data format shall define the syntax for a review
element.  The purpose of an review element is to declare the status
value of an entity as determined by an informal review.  The review
information shall include: entity reviewed, date of review, status
determined by the review, and a description (not to exceed one
paragraph) of the review.
