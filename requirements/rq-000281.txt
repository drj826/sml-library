>>>requirement.rq-000281

title:: Publish A Static HTML Page For Each Document Section

is_part_of:: rq-000001          # Publish an SML Document to Static HTML

type:: functional

description:: The software shall publish a static HTML page for each
document [g:SML:SECTION].  The purpose of this requirement is to
improve document readability by keeping the size of each page small
and to keep the size of each HTML file small to speed browser page
rendering.  Chapters and appendices are considered types of sections.
