title:: Define the Syntax for the Start of a Named Division

type:: data format

is_part_of:: rq-000326

description:: The data format shall define the syntax for the start of
a named division.  Named divisions represent document structures
composed of other (sub) divisions and content blocks.  The start
division syntax shall capture the division name and ID.

outcome::2016-02-29:rq-000361:green: This requirement is
