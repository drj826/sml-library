>>>requirement.rq-000048

title:: Be Encodable Using 7-Bit ASCII

revision:: $Revision: 10973 $

date:: $Date: 2012-10-27 14:46:35 -0600 (Sat, 27 Oct 2012) $

author:: $Author: don.johnson $

priority:: routine

is_part_of:: rq-000107

type:: data format

description:: The data format shall be encodable using 7-bit ASCII
(a.k.a. ISO/IEC 646:1991 ISO 7-bit coded character set for information
exchange).  This means that all document content and markup be
representable using 7-bit ASCII encoding.  The purpose of this
requirement is to keep the data format simple to maximize the number
of tools capable of processing documents written in the format.

index:: [id:rq-000048]; 7-bit ASCII; ISO/IEC 646:1991
