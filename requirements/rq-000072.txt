>>>requirement.rq-000072

title:: Represent Script Statements

priority:: routine

is_part_of:: rq-000009

type:: data format

description:: The data format shall have the capability to represent
executable script statements to be executed by the publishing
application.  The purpose of this requirement is to enable the
publishing application to execute scripts that dynamically generate
document content at publish time.

index:: [id:rq-000072]; scripts

The data format must provide the capability to represent the following
variations on script-generated content:

+ ~~hide~~ -- Run the script but do not show the output in the
  published rendition.
