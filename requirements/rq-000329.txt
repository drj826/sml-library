description:: The software shall have the capability to [g:SML:format]
a document block according to a template.

###comment

Templates should be organized by name and style.  For instance there
might be a 'default' style for a 'title' element (block).  The
template should be a function of:

1. block name (i.e. title)

2. style name (i.e. default)

3. path 

  examples:

  document/title
  document/section/section/title
  document/section/paragraph
  document/section/table/table-row/table-cell/list-item
