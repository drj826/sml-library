>>>requirement.rq-000453

title:: Publish a Static HTML Library Ontology Page

type:: functional

is_part_of:: rq-000443          # Publish an SML Library to Static HTML

description:: The software shall publish a static HTML library
[g:SML:ontology] page. The purpose of the ontology page is to inform a
reader of the entities allowed to exist in the library, the properties
of those entities, and how the entities are allowed to relate to one
another.  The ontology page should allow the reader to discover the
concepts and relationships library documents are intended to convey.

index:: rq-000453; ontology

outcome::2016-03-25:rq-000453:green: This requirement is completely
defined. (Don Johnson)
