title:: Define the Syntax for the End of a Document Section

outcome::2016-03-02:rq-000364:red: I'm not convinced this requirement
is necessary.  The technical solution I've chosen has ~~no~~ explicit
syntax for the end of a document section.  Instead, the end of a
document section is implied by either (1) the beginning of the next
section, or (2) the end of the document. (Don Johnson)
