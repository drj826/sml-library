>>>requirement.rq-000386

title:: Define the Syntax for an Attribute Element

type:: data format

is_part_of:: rq-000372

description:: The data format shall define the syntax for an attribute
element.  The purpose of an attribute element is to define a property
name and value that are not declared in the ontology.  This gives
authors the flexibility to add properties without changing the library
ontology. The syntax shall capture the attribute name, an optional
namespace, and the attribute value. The namespace is a string of
characters that disambiguates the attribute definition from other
definitions of the same attribute.

index:: rq-000386; attribute element; element!attribute

outcome::2016-03-05:rq-000386:green: This requirement is
complete. (Don Johnson)
