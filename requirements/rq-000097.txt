>>>requirement.rq-000097

title:: Represent a Source Citation

priority:: routine

is_part_of:: rq-000120

type:: data format

description:: The data format shall have the capability to represent a
source citation.  In other words, the data format must have the
capability to indicate when information originates from a
bibliographic source.  The data format must allow source citations to
span newline characters.

index:: rq-000097; source citations
