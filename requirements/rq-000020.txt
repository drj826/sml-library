title:: Represent an Inline Cross Reference

revision:: $Revision: 10973 $

date:: $Date: 2012-10-27 14:46:35 -0600 (Sat, 27 Oct 2012) $

author:: $Author: don.johnson $

priority:: routine

is_part_of:: rq-000334

type:: data format

associated:: rq-000021

associated:: rq-000022

description:: The data format shall have the capability to represent
an inline cross reference.  An inline cross reference enables authors
to direct the reader's attention to another part of the document.  The
cross referencing capability must enable authors to cross reference
specified document elements.

index:: [id:rq-000020]; cross references

The data format must enable authors to cross reference the following
elements of document content:

+ sections

+ tables

+ figures

+ attachments

+ listings

+ exercises

+ presentation slides

+ quotations

+ demonstrations

+ key-points listings

+ problems

+ solutions

+ tests

+ results

+ tasks

+ roles
